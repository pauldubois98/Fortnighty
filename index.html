Aim

Inspiration
Our task was set out by the prestigious American Express. We were tasked by recreating a modern day website as if it were made in the 90's . As avid gamers, we all thought what's the best game currently out there on the market? Exactly Red Dead Redemption 2. But due to the violent and graphic nature of this 18+ year rated game, we recreated the much more family friendly version, Fortnite.

What it does?
It is a website that plays a more 90's retro (and arguably, better and harder to win) version of Fortnite.

How we built it?
We built it entirely from scratch (not the MIT one) by creating the graphics in Photoshop, website in HTML and game in JavaScript. Essentially, we didn't steal any copyrighted material so if Epic Games' attorney would like to contact us, please redirect all emails to the sender.

Challenges we ran into
We ran into many many bugs that we are currently working on to remove, however we are trying to find the optimal ratio of removing bugs and removing bugs that do not create 3 new ones.

What's next for FortNightly?
The future of FortNightly is the 00's version and launching our IPO with the dotcom bubble. Surely this will never crash.
